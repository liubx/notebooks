# 👋 欢迎来到 Bingxin's Vault

这是你的个人知识管理空间，使用 PARA + Zettelkasten 方法论组织内容。

## 📁 目录结构

- **0-Daily** — 每日笔记、周回顾、月回顾
- **1-Projects** — 进行中的项目（工作项目、个人项目）
- **2-Areas** — 持续关注的领域（工作、生活）
- **3-Resources** — 知识库和参考资料
- **4-Archives** — 已完结的内容

## 🚀 快速开始

1. 在 `0-Daily` 中写今天的日记
2. 在 `1-Projects` 中创建你正在做的项目
3. 在 `2-Areas` 中积累工作资料和生活笔记
4. 在 `3-Resources` 中收集有用的知识卡片

## 📋 任务管理

查看 [[任务总览]] 了解所有待办事项。

任务使用 Obsidian Tasks 插件管理，格式：

```
- [ ] 任务内容 📅 2026-03-25 #task/work #project/项目名
```
