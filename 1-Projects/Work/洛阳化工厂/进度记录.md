## 数据库 jsonb 字段改造

| 表名 | 字段 | 备注 |
|------|------|------|
| t_module_type | config jsonb | 修改 ModuleTypeEntity |
| t_project | config jsonb | 修改 ProjectEntity |
| t_alarm_type | config jsonb | 修改 AlarmType && AlarmTypeEntity |
| t_fence | config jsonb, properties jsonb | 修改 Fence && FenceEntity |
