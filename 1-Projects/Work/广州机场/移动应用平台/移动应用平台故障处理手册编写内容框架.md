- 移动应用平台无法登陆
- 移动应用平台App无法登陆

## 数据库相关

主从备份架构，基于docker

### 数据库相关服务器

- 10.120.78.43 (A域)
- 10.120.78.242 (A域)
- 10.120.79.247 (B域)
- 10.120.78.35 (B域)

### 查看数据库状态

```bash
docker ps
docker exec -it timescaledb-ha patronictl -c /home/postgres/postgres.yml list
```

### 停止数据库

```bash
docker stop timescaled-ha
docker stop etcd
```

### 启动数据库

```bash
docker start timescaled-ha
docker stop etcd
```

### 重新运行数据库相关容器

```bash
cd /opt/reliablesense/timescaledb
docker-compose up -d

cd /opt/reliablesense/etcd
docker-compose up -d
```
