# **故障场景**
- 综合定位平台无法登陆
- 巡检App无法登陆
- 定位终端无法定位
- GNSS服务器（问场景）<mention-user id="ou_c0539b45af4070465c5fe10869002f89"/>

## 数据库相关
主从备份架构，基于docker
### 数据库相关服务器
- 10.120.79.4 (A域)
- 10.120.79.102 (A域)
- 10.120.78.74 (B域)
- 10.120.79.197 (B域)
### 查看数据库状态
```plaintext
docker ps
docker exec -it timescaledb-ha patronictl -c /home/postgres/postgres.yml list
```

### 停止数据库
```plaintext
docker stop timescaled-ha
docker stop etcd
```

### 启动数据库
```plaintext
docker start timescaled-ha
docker stop etcd
```

### 重新运行数据库相关容器
```plaintext
cd /opt/reliablesense/timescaledb
docker-compose up -d

cd /opt/reliablesense/etcd
docker-compose up -d
```


# **操作手册**
- 重启集群/集群内服务相关指令
- 重启数据库相关指令
