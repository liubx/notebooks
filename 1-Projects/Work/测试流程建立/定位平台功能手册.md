# 室内外定位平台功能手册

> 本手册完整描述定位平台所有功能模块、操作路径、页面元素与业务规则，供测试用例生成与AI自动化操作使用。

---

## 1. 系统概述

室内外定位平台是集定位、导航、地图、信息展示为一体的数字化管理平台。基于精准定位技术，实现定位展示、地图导航、轨迹追踪、围栏报警、影像联动、大数据分析等功能。

### 1.1 系统架构

- 前端：Web管理平台 + 移动端APP + 导航小程序
- 后端：定位引擎 + 数据推送服务（MQTT/UDP/Webhook/HTTP）
- 设备层：定位标签（Tag）、基站（Anchor）、摄像头

### 1.2 功能模块总览

| 模块 | 入口路径 | 功能简述 |
|------|----------|----------|
| 登录 | 系统首页 | 账户密码登录，含验证码校验 |
| 总览 | 总览 | 项目内设备与对象数量汇总 |
| 实时地图 | 实时地图 | 地图可视化查看定位对象、设备、围栏实时状态 |
| 定位对象管理 | 定位对象管理 | 定位对象的增删改查、分组、标签管理 |
| 设备管理 | 设备管理 | 定位标签、基站、摄像头的管理 |
| 地图管理 | 地图管理 | 建筑、楼层、地图、兴趣点、文字管理 |
| 围栏管理 | 围栏管理 | 电子围栏的创建、绘制、启停管理 |
| 告警管理 | 告警管理 | 当前告警与历史告警的查看与筛选 |
| 数据分析 | 数据分析 | 在线时长、围栏分析、告警统计、热力图、轨迹 |
| 数据大屏 | 数据大屏 | 自定义监控大屏 |
| 通知管理 | 通知管理 | 通知发布与查看 |
| 公司管理 | 公司管理（超级管理员） | 公司的增删改查 |
| 用户管理 | 用户管理 | 用户的增删改查、角色分配 |
| 项目管理 | 项目管理 | 项目的增删改查、权限设置 |

## 2. 角色与权限体系

### 2.1 角色定义

| 角色 | 创建者 | 权限范围 |
|------|--------|----------|
| 超级管理员 | 平台分配 | 管理所有公司、项目、用户；不能进入项目查看业务数据 |
| 公司管理员 | 超级管理员创建 | 管理所属公司的所有项目（增删改查）；管理公司用户；设置项目权限与角色 |
| 项目管理员 | 公司管理员/超级管理员指定 | 查看、使用、编辑项目所有功能 |
| 项目成员 | 公司管理员/超级管理员指定 | 查看、使用项目所有功能，无编辑权限 |
| 公司普通用户 | 超级管理员创建 | 查看已加入的项目 |

### 2.2 权限控制

- 菜单权限：控制用户可见的功能模块
- 按钮权限：控制用户可执行的操作（新建、编辑、删除等）
- 权限设置路径：项目管理 → 项目权限 → 选择项目 → 选择用户 → 点击【权限】

## 3. 登录模块

### 3.1 登录页面

- 访问路径：浏览器打开系统地址，自动跳转登录页
- 登录方式：账户密码登录

### 3.2 登录表单

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 用户名 | 文本输入框 | 是 | 由管理员分配 |
| 密码 | 密码输入框 | 是 | 默认密码由管理员告知 |
| 验证码 | 文本输入框 | 是 | 输入右侧验证码图片中的字符 |

### 3.3 验证码规则

- 不区分大小写
- 输入错误提示"验证码错误"，验证码图片自动刷新
- 点击验证码图片可手动刷新

### 3.4 登录操作流程

1. 输入用户名
2. 输入密码
3. 查看验证码图片，输入验证码
4. 点击【登 录】按钮
5. 成功：跳转至项目列表页面，右上角显示当前用户名称
6. 失败：页面顶部提示"用户名或密码错误"

### 3.5 异常场景

| 场景 | 预期行为 |
|------|----------|
| 用户名为空 | 提示必填 |
| 密码为空 | 提示必填 |
| 验证码为空 | 提示必填 |
| 用户名或密码错误 | 提示"用户名或密码错误" |
| 验证码错误 | 提示"验证码错误"，验证码自动刷新 |

## 4. 总览模块

### 4.1 功能说明

对项目内配置的定位标签、定位对象、围栏、基站、摄像头进行数量汇总展示。

### 4.2 页面元素

| 元素 | 说明 |
|------|------|
| 定位标签数量 | 项目内定位标签总数 |
| 定位对象数量 | 项目内定位对象总数 |
| 围栏数量 | 项目内电子围栏总数 |
| 基站数量 | 项目内基站总数 |
| 摄像头数量 | 项目内摄像头总数 |
| 【详情】按钮 | 点击跳转至对应模块详情页 |

## 5. 公司管理（超级管理员）

### 5.1 入口路径

超级管理员 → 公司管理 → 公司列表

### 5.2 新建公司

- 操作：点击【新建】按钮 → 打开新建公司弹窗 → 填写公司基本信息 → 确定
- 关键字段：公司ID（唯一标识，创建后不可修改）

### 5.3 修改公司

- 操作：点击【修改】按钮 → 修改除公司ID外的信息 → 确定

### 5.4 删除公司

- 单个删除：点击【删除】按钮
- 批量删除：勾选多个公司 → 点击【批量操作-删除选中】
- 业务规则：删除公司后，该公司下的全部信息同步删除

---

## 6. 用户管理

### 6.1 入口路径

用户管理 → 用户列表

### 6.2 新建用户

- 操作：点击【新建】按钮 → 打开新建用户弹窗 → 填写用户信息 → 确定
- 必填信息：用户名、所属公司、角色
- 业务规则：用户名创建后不可修改；员工凭用户名与默认密码登录

### 6.3 修改用户

- 操作：点击【编辑】按钮 → 修改除用户名外的信息 → 确定

### 6.4 删除用户

- 单个删除：点击【删除】按钮，用户从平台删除，无法再登录
- 批量删除：勾选多个用户 → 点击【批量操作-删除选中】

### 6.5 移除用户

- 操作：点击【移除】按钮
- 业务规则：将用户移出公司，但保留用户信息（与删除不同）

## 7. 项目管理

### 7.1 项目列表

- 入口路径：项目管理 → 项目列表

#### 7.1.1 新建项目

- 操作：点击【新建项目】→ 打开新建项目弹窗 → 填写项目基本信息 → 确定
- 业务规则：项目所属公司与ID创建后不可修改

#### 7.1.2 修改项目

- 操作：点击【项目信息】→ 弹窗中修改项目名称和描述 → 确定

#### 7.1.3 删除项目

- 操作：点击【项目信息】→ 弹窗中点击【删除】按钮

### 7.2 项目权限

- 入口路径：项目管理 → 项目权限

#### 7.2.1 添加用户到项目

操作步骤：
1. 在左侧项目栏中选择指定项目
2. 在右侧用户列表中点击【添加】按钮 → 打开添加弹窗
3. 输入要添加的用户名称，选择角色（项目管理员/项目成员）
4. 点击【权限】→ 设置菜单及按钮权限

#### 7.2.2 移除用户

- 单个移除：点击【移除】按钮
- 批量移除：勾选多个用户 → 点击【批量操作-移除选中】

## 8. 定位对象管理

### 8.1 功能概述

统一管理全部定位对象，支持分组与标签分类管理，可查看对象详情、告警信息、运动轨迹。

### 8.2 标签配置

- 入口路径：定位对象管理 → 标签配置

| 操作 | 方式 | 说明 |
|------|------|------|
| 新增标签 | 点击【新标签】→ 输入标签名称 | 建议名称不重复 |
| 修改标签 | 双击已有标签名称 → 修改 | 光标聚焦后可编辑 |
| 删除标签 | 点击【删除×】按钮 | 绑定该标签的定位对象标签位同时清空 |

### 8.3 群组管理

- 入口路径：定位对象管理 → 定位对象群组列表

#### 8.3.1 创建群组

- 首次创建：点击【新建群组】按钮
- 已有群组时：点击【新建】按钮
- 弹窗填写：群组名称、标签（绑定群组的对象自动绑定该标签）

#### 8.3.2 修改群组

- 操作：点击编辑按钮 → 弹窗修改群组基本信息

#### 8.3.3 删除群组

- 操作：点击删除按钮
- 业务规则：绑定该群组的定位对象群组信息同时清空

#### 8.3.4 群组成员管理

- 操作：点击【成员管理】
- 移入成员：成员自动绑定该群组
- 移除成员：成员自动解绑该群组

### 8.4 定位对象列表

- 入口路径：定位对象管理 → 定位对象列表

#### 8.4.1 新建定位对象

三种创建方式：

| 方式 | 说明 |
|------|------|
| 系统自动创建 | 用户通过小程序访问后自动创建 |
| 手动单个新增 | 点击【新建】→ 填写定位对象信息 |
| 手动批量导入 | 点击【批量操作-批量导入】→ 上传Excel文件 |

#### 8.4.2 单个新增定位对象表单

| 字段 | 说明 |
|------|------|
| 基本信息 | 定位对象名称等 |
| 标签 | 分类管理，可在实时地图、数据分析中按标签查询 |
| 群组 | 分类管理，可在实时地图中按群组查询；可快速绑定防撞规则 |
| 动态电子围栏 | 开启后围栏跟随对象移动，在半径内产生告警 |
| 更新间隔 | 按设定间隔更新位置信息 |
| 防撞警告 | 防撞相关参数设置 |

#### 8.4.3 定位对象列表操作

| 操作 | 方式 | 说明 |
|------|------|------|
| 搜索 | 按定位对象名称搜索 | — |
| 解绑 | 点击【解绑】 | 解除定位对象与定位标签的绑定关系 |
| 删除 | 点击【删除】或【批量操作-删除选中】 | 支持单个和批量 |
| 下载选中 | 勾选后点击【批量操作-下载选中】 | 导出为Excel |
| 下载全部 | 取消勾选后点击【批量操作-下载全部】 | 导出全量为Excel |

#### 8.4.4 定位对象详情

- 操作：点击【详情】按钮
- 展示内容：基本信息、绑定的定位标签信息、告警信息、移动轨迹
- 可执行操作：解绑、重新编辑、删除定位标签

## 9. 设备管理

### 9.1 功能概述

统一管理定位标签、基站、摄像头设备，支持设备类型配置、在线状态监控、电量监控。

### 9.2 设备配置

- 入口路径：设备管理 → 设备配置

#### 9.2.1 定位标签类型配置

| 操作 | 方式 | 说明 |
|------|------|------|
| 创建类型 | 点击"定位标签tab" → 点击【新建】 | 填写类型信息 |
| 修改类型 | 点击【编辑】 | 重新填写类型信息 |
| 删除类型 | 点击【删除】或【批量操作-删除选中】 | 绑定该类型的标签将清空类型和地址 |
| 停用/启用 | 点击【停用/启用】开关 | 停用后的类型无法被定位标签使用 |

#### 9.2.2 基站类型配置

| 操作 | 方式 | 说明 |
|------|------|------|
| 创建类型 | 点击"基站tab" → 点击【新建】 | 填写类型信息 |
| 修改类型 | 点击【编辑】 | 重新填写类型信息 |
| 删除类型 | 点击【删除】或【批量操作-删除选中】 | 绑定该类型的基站将清空类型 |
| 停用/启用 | 点击【停用/启用】开关 | 停用后的类型无法被基站使用 |

### 9.3 定位标签管理

- 入口路径：设备管理 → 定位标签管理

#### 9.3.1 创建定位标签

- 操作：点击【新建】→ 打开新建定位标签弹窗
- 表单字段：类型（从设备配置中选择）、模组信息（根据类型配置各模组地址）

#### 9.3.2 修改定位标签

- 操作：点击【详情】→ 点击【编辑】→ 修改信息

#### 9.3.3 定位标签列表操作

| 操作 | 方式 |
|------|------|
| 解绑 | 点击【解绑】，解除与定位对象的绑定 |
| 删除 | 点击【删除】或【批量操作-删除选中】 |
| 下载选中 | 勾选后点击【批量操作-下载选中】，导出Excel |
| 批量导入 | 点击【批量操作-批量导入】，上传Excel |

#### 9.3.4 定位标签详情

- 展示内容：基本信息、绑定的定位对象信息、操作日志
- 可执行操作：解绑、删除、修改、跳转至定位对象详情

### 9.4 基站管理

- 入口路径：设备管理 → 基站管理

#### 9.4.1 创建基站

- 操作：点击【新建】→ 打开新建基站弹窗
- 表单字段：

| 字段 | 说明 |
|------|------|
| 类型 | 从设备配置中选择 |
| 地址 | 输入基站MAC地址 |
| 坐标 | 在右侧地图中双击位置自动填充；可再次双击更新 |

#### 9.4.2 修改基站

- 操作：点击【详情】→ 点击【编辑】→ 修改信息

#### 9.4.3 基站列表操作

| 操作 | 方式 |
|------|------|
| 删除 | 点击【删除】或【批量操作-删除选中】 |
| 下载选中 | 勾选后点击【批量操作-下载选中】，导出Excel |
| 批量导入 | 点击【批量操作-批量导入】，上传Excel |

#### 9.4.4 基站详情

- 展示内容：基本信息、基站电量、基站图片、基站位置
- 可执行操作：删除、上传图片

### 9.5 摄像头管理

- 入口路径：设备管理 → 摄像头管理

#### 9.5.1 创建摄像头

- 操作：点击【新建】→ 打开新建摄像头弹窗
- 坐标：在右侧地图中双击位置自动填充；可再次双击更新

#### 9.5.2 修改摄像头

- 操作：点击【详情】→ 点击【编辑】→ 修改信息

#### 9.5.3 摄像头列表操作

| 操作 | 方式 |
|------|------|
| 删除 | 点击【删除】或【批量操作-删除选中】 |
| 下载选中 | 勾选后点击【批量操作-下载选中】，导出Excel |

#### 9.5.4 摄像头详情

- 展示内容：基本信息、摄像头实况、摄像头位置
- 可执行操作：删除

## 10. 地图管理

### 10.1 功能概述

为项目区域内的建筑设置楼层，为楼层设置地图，为地图设置兴趣点和文字标注。

### 10.2 操作总流程

1. 新增建筑 → 2. 进入建筑，新增楼层 → 3. 文件管理，上传地图图片 → 4. 地图编辑，配置兴趣点和文字

### 10.3 建筑管理

- 入口路径：地图管理

#### 10.3.1 新增建筑

- 操作：点击【新增建筑】→ 打开弹窗 → 填写建筑信息
- 编辑区域：点击【编辑区域】→ 在地图中单击点选位置描绘边界 → 自动闭环 → 按回车键完成
- 删除区域：点击【删除区域】→ 清空描绘的区域和点
- 重置：清空区域并恢复默认地图视角

#### 10.3.2 编辑建筑

- 操作：点击【编辑信息】→ 打开编辑弹窗

#### 10.3.3 建筑列表

- 支持按名称搜索
- 支持切换地图视图/列表视图
- 支持【隐藏】暂不使用的建筑

### 10.4 楼层管理

- 入口路径：地图管理 → 点击【进入建筑】

#### 10.4.1 新增楼层

- 操作：点击左侧楼层树中的【新建】→ 打开添加楼层弹窗
- 楼层等级规则：一层=1，负一层=-1，二层=2，以此类推
- 编辑区域：同建筑区域编辑方式

#### 10.4.2 修改楼层

- 操作：点击左侧楼层树中的【详情】→ 打开修改楼层信息弹窗

### 10.5 地图管理

#### 10.5.1 创建地图

- 操作：在左侧楼层树中选择楼层 → 点击【添加】→ 打开新建地图弹窗

#### 10.5.2 修改地图

- 操作：点击【编辑】→ 打开修改地图信息弹窗

#### 10.5.3 文件管理

- 操作：选择楼层的地图 → 点击【文件管理】→ 上传地图图片
- 删除文件：点击【删除】（若文件已被使用且已配置兴趣点、文字，将同步删除所有配置）

#### 10.5.4 地图图片编辑

- 操作：点击【地图编辑】→ 选择"图片编辑"tab → 点击【图片选择】→ 选择已上传文件 → 确定

### 10.6 兴趣点编辑

- 入口：地图编辑 → "兴趣点编辑"tab

#### 10.6.1 新增兴趣点

操作步骤：
1. 选择图标（公共图标或自定义上传.png格式）
2. 选择兴趣点类型
3. 输入兴趣点名称
4. 输入离地高度
5. 在地图中双击鼠标标注位置

#### 10.6.2 管理自定义图标

- 操作：点击【添加】→ 打开自定义图标管理弹窗
- 删除图标：单击某图标即可删除

#### 10.6.3 编辑兴趣点类型

## 11. 围栏管理

### 11.1 功能概述

灵活设置电子围栏，监控区域内的人员和资产，支持多种围栏类型和告警规则。

### 11.2 创建围栏

#### 11.2.1 围栏基础属性

- 操作：围栏管理 → 点击【新建】→ 打开新建围栏弹窗

| 字段 | 说明 |
|------|------|
| 围栏类型 | 支持多种空间、时间维度的围栏 |
| 对象额度 | 围栏类型为"限制数额"时生效，对象数额不在范围内产生告警 |
| 限制时间 | 围栏类型为"限时进入"或"限时离开"时生效，超时产生告警 |
| 围栏状态 | 已启用/已停用，只有已启用的围栏才产生告警 |
| 激活时间 | 只在规定的激活时间内产生告警 |
| 围栏应用范围 | ①除列表以外的所有项：对设置对象以外的所有对象告警；②仅用于列表：只对设置的对象告警 |
| 分组 | 可使用分组快速添加定位对象 |

#### 11.2.2 绘制围栏边界

操作步骤：
1. 填写基础属性后点击【确定】→ 打开围栏详情界面
2. 在室外地图中选择所属建筑 → 进入室内地图 → 选择围栏所属楼层
3. 点击【编辑围栏】→ 进入编辑模式
4. 在地图中单击点选位置描绘边界 → 自动闭环 → 按回车键完成

| 操作 | 说明 |
|------|------|
| 编辑围栏 | 单击点选描绘边界 |
| 删除围栏 | 点击【删除围栏】清空区域和点 |
| 撤销 | 回到上一步 |
| 退回 | 回到下一步（重做） |

### 11.3 围栏列表

- 入口路径：围栏管理
- 展示全部围栏，可设置启用/停用状态

| 操作 | 方式 |
|------|------|
| 编辑围栏 | 点击【编辑】→ 围栏详情界面 → 修改属性或重新绘制边界 |
| 删除围栏 | 列表中【批量操作-删除选中】或详情界面【删除】 |

- 业务规则：围栏删除后，该围栏产生的告警信息全部移入历史告警

## 17. 数据对接（API）

### 17.1 概述

系统支持通过API接口获取定位标签的UID、坐标、电量、工作状态及实时定位信息。

- 数据格式：JSON
- 接口协议：MQTT、UDP、Webhook、HTTP
- 支持平台：Ubuntu、Windows、macOS

### 17.2 标签定位数据

消息类型：`Pos`

| 字段 | 类型 | 说明 |
|------|------|------|
| deviceName | string | 设备名，系统内唯一 |
| name | string | 消息类型，固定为"Pos" |
| uid | string | 设备UID |
| data.mapId | int | 定位所在楼层ID |
| data.pos | float[3] | 定位坐标 [x, y, z] |
| data.noiseRadius | float | 定位噪声，值越大置信度越低 |
| data.time | long | 定位时间戳（Unix毫秒） |
| data.vel | float[3] | 速度 [x, y, z] |
| data.rxAnchorSn | string | 最强接收信号对应基站UID |
| data.rxRssi | float | 最强接收信号强度（dBm） |
| data.fences | array | 标签所在的电子围栏列表 |
| data.optPoses | float[][3] | 优化输出位置（含边界、墙约束） |

### 17.3 数据推送协议

#### 17.3.1 MQTT推送

| 参数 | 说明 |
|------|------|
| address | 连接地址，格式：`协议://用户名:密码@主机名:端口` |
| clientId | 客户端标识，可为空 |
| topic | MQTT主题名 |

支持协议：mqtt、tcp、ws（不加密）；mqtts、tcps、wss（加密）

#### 17.3.2 UDP推送

| 参数 | 说明 |
|------|------|
| address | 目标地址，格式：`IP或域名:端口` |
| localAddress | 本地IP地址（可选），多网卡环境下指定发送网卡 |
