# 集群部署相关

### Nodeport Service

```sql
apiVersion: v1
kind: Service
metadata:
  labels:
    k8s-app: reliablesense-demo-huayuen-uwb
    qcloud-app: reliablesense-demo-huayuen-uwb
  name: reliablesense-demo-huayuen-uwb-node-port
  namespace: mid-dev
spec:
  clusterIP: 192.168.255.45
  externalTrafficPolicy: Cluster
  ports:
  - name: 6666-6666-tcp
    nodePort: 30600
    port: 6666
    protocol: TCP
    targetPort: 6666
  selector:
    k8s-app: reliablesense-demo-huayuen-uwb
    qcloud-app: reliablesense-demo-huayuen-uwb
  sessionAffinity: None
  type: NodePort
```

  revisionHistoryLimit: 10
