## 用户相关接口里返回的projects不统一

### 超级管理员

- [ ]  有的接口返回了所有的项目，projects项目里不会包含权限和角色
    - GET /api/sessions
    - GET /api/users/{id}
- [ ]  有的接口返回当前用户所加入的项目，projects项目里会包含权限和角色
    - GET /api/users
    - GET /api/users/pages
    

### 公司管理员

- [ ]  有的接口返回了该公司的所有项目，projects项目里不会包含权限和角色
    - GET /api/sessions
    - GET /api/users/{id}
- [ ]  有的接口返回当前用户所加入的项目，projects项目里会包含权限和角色
    - GET /api/users
    - GET /api/users/pages

### 公司普通用户

- [ ]  有的接口返回当前用户所加入的项目，projects项目里包含权限和角色
    - GET /api/sessions
    - GET /api/users/{id}
- [ ]  接口无权限访问（没问题）
    - GET /api/users
    - GET /api/users/pages
