# 监听捉包转发

```jsx
// 发送udp数据
echo "hello" | socat - UDP4-DATAGRAM:152.136.147.76:7890
echo "hello" | socat - UDP4-DATAGRAM:127.0.0.1:32500

// 抓包
tcpdump port 9000 -v -XX -n
tcpdump udp port 9000 -v

tcpdump -i ens192 udp port 32500 -v

// 监听
socat TCP4-LISTEN:80 STDOUT

socat UDP4-LISTEN:9000 STDOUT
```
