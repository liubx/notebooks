# 导航相关

这个组件我想进一步的封装

- 把这个做成一个通用的多楼层线段显示的组件，只需要显示线，不需要显示图标，图标真正使用的时候用@/modules/Marker就行,好像没必要封装到组件这里， 传进来的data可以按照下面的格式传入

```
[
  ...,
  {
    floor: 1,
    spaceId: 1,
    mapId: 1,
    data: [
      ...,
      {
        longitude: xxx,
        latitude: xxx,
        altitude: xxx
      }
    ]
  },
  {
    floor: 1,
    spaceId: 1,
    mapId: 1,
    data: [
      ...,
      {
        longitude: xxx,
        latitude: xxx,
        altitude: xxx
      }
    ]
  }
]
```

- data数组里放每一段线段，可以调用 @/modules/Line 来显示画到
- 数组间相邻的线段，需要连接起来，spaceId 不一样的话不需要连接起来
