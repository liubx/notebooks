# 不要孩子影响

Complete: No
Section: 要不要小孩

正面影响：

1. 我们可以补做孕检，做更多的准备，尽量保证孩子的健康 （健康+）
2. 我们能有更多两个世界的时间，去旅游（家庭+）
3. 能保证拿到加拿大的PR，可以以很低费用到加拿大生孩子（经济+）
4. 能积累更多的财富，保证以后有孩子的时候，能给孩子提供更好的环境 （经济+）

负面影响：

1. 失去了一个生命 （生命-）
2. 影响个人身体和心理健康，影响以后生育 （健康-）
3. 家庭意见不和，发生争执 （家庭-）
4. 我们要晚一点才能有自己的孩子 （时间-）
5. 可能没办法陪你做人流手术，要的话也要支出很多费用 （家庭-,经济-）
