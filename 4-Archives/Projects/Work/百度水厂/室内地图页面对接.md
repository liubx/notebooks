# 地图地址
```plaintext
http://10.16.15.243/map
```

# 测试项目密钥
```plaintext {wrap}
appId: cnnc_qshdz
appSecret: 214e6658-54e9-47f3-9aae-ba4d4ee17c78

http://10.16.15.243/map/?appId=cnnc_qshdz&appSecret=214e6658-54e9-47f3-9aae-ba4d4ee17c78
```

作为参数传入url，指定该密钥后，地图页面默认显示该项目的地图数据，包括 建筑，楼层，地图，POI 等数据信息
# 测试项目密钥
```plaintext {wrap}
floor: 1
spaceId: 1

http://10.16.15.243/map/?appId=cnnc_qshdz&appSecret=214e6658-54e9-47f3-9aae-ba4d4ee17c78&spaceId=1&floor=1
```

作为参数传入url，可通过参数控制默认进入的建筑和楼层
# 获取定位对象id
```plaintext {wrap}
trackableId：xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx

http://10.16.15.243/map/?appId=cnnc_qshdz&appSecret=214e6658-54e9-47f3-9aae-ba4d4ee17c78&trackableId=81746448-0404-4cbf-9a35-2012e381689e
```

作为参数传入url，绑定该定位对象作为手机的当前位置，该定位对象可在定位平台手动新建/获取，或者通过Api接口新建/获取
# 完整地址示例
```plaintext {wrap}
http://10.16.15.243/map/?appId=cnnc_qshdz&appSecret=214e6658-54e9-47f3-9aae-ba4d4ee17c78&spaceId=1&floor=1&trackableId=81746448-0404-4cbf-9a35-2012e381689e
```
