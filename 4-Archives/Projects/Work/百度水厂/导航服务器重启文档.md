# 导航服务器重启文档

启动 putty 软件

输入堡垒机IP地址 10.16.68.92 并连接

输入用户名及密码，密码为用户密码+空格+6位随机码

登录导航服务器

```
ssh root@10.16.15.243
```

回车后输入以下密码 xxxxxx (输入内容不会显示出来)

输入以下指令，更新程序

```bash
// 暂停k3s服务
systemctl stop k3s
// 检查k3s服务是否已经停止
systemctl status k3s
// 重启服务器
reboot
```

等待服务器重启，各个服务会自动重启
