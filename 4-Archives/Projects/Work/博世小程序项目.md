# 博世 & 莱讯 & 南讯集成

开发了一个小程序应用，对接定位平台。

## 集成问题记录

### 基础鉴权
- 调用登录接口后获取 token，通过 token 和项目 id 调用其他接口

### 地图基础数据
- 根据 mapId 获取室内地图
- 地图基础数据变更后的同步方式

### 实时定位获取
- 小程序 SDK 初始化传入用户 openID
- openID 与 coreaiot_generate_service_uuids 的映射关系
- 通过 SDK 回调函数接收位置更新

### 用户轨迹数据
- 需要按时间段同步用户轨迹信息的接口
