# 产品概述

基于红点定位（Redpoint Positioning）的信息服务系统，围绕仓储行业叉车作业能效管理和安全管理需求，涵盖人员、车辆、设备、资产等多方面的综合智能管理解决方案。

# 产品功能

- 实时地图：实时地图、热力图、轨迹图
- 路径规划
- 叉车管理
- 驾驶员管理
- 设备管理：标签管理、基站（锚节点）管理
- 围栏管理
- 告警管理
- 系统管理
- 账号管理

# 防碰撞告警与动态电子围栏

## 防撞告警

- 当广播者附近出现广播者或监听者时，双方同时告警
- 监听者之间不会告警
- 壁挂式告警灯：静态时声光告警，动态时只发光
- 推荐设置：广播者=叉车，监听者=驾驶员（人员）
- 可同时加入多个群组，但只能有一个群组有防撞告警设置
- 角色为监听者时，只可对角色为「广播者」的定位对象不返回报告
- 添加定位对象时选择「群组」，防撞告警设置处出现「跟随所属群组」选项

## 动态电子围栏（Buffer）

和静态电子围栏类似，半径中只要出现任何设备，对应设备都会告警。只需设置一个标签，一般用于给会动的货架使用。


# 数据字段定义

## 叉车管理

| 分类 | 字段 |
|------|------|
| 基本信息 | 编号、品牌、型号、内燃/充电、状态（行驶/静止）、来源（外包/自购）、群组、总使用时间、备注 |
| 绑定信息 | 标签编号、MAC地址、电量、绑定时间、最新上报时间、类型、操作员 |
| 工作记录 | 开机时间、关机时间、总时长、运单数量、越界、超速、急刹、里程、事故 |
| 驾驶员相关 | 姓名、工号 |
| 动态围栏 | 围栏半径、应用对象 |
| 高级设置 | 默认高度、更新间隔 |
| 防撞设置 | 角色、缓冲距离 |

## 驾驶员管理

| 分类 | 字段 |
|------|------|
| 基本信息 | 姓名、状态（驾驶中/非驾驶中）、群组、部门、工号、电话、驾驶证编号 |
| 绑定信息 | 标签编号、MAC地址、电量、录入时间、最新上报时间、类型、操作员 |
| 工作记录 | 开始时间、结束时间、总时长、运单数量、越界、超速、急刹、里程、事故 |
| 叉车相关 | 编号、里程 |
| 动态围栏 | 围栏半径、应用对象 |
| 高级设置 | 默认高度、更新间隔 |
| 防撞设置 | 角色、覆盖距离、覆盖角度 |

## 标签管理

| 分类 | 字段 |
|------|------|
| 基本信息 | 标签编号、MAC地址、电量、录入时间、最新上报时间、类型 |
| 绑定信息 | 叉车编号/驾驶员编号（姓名）、操作记录（解绑/绑定） |

## 基站管理

基本信息：MAC地址、名称、位置、类型（桥节点/锚节点）、状态（在线/离线）

## 群组

| 分类 | 字段 |
|------|------|
| 基本信息 | 名称、成员数量 |
| 防撞设置 | 角色、缓冲距离、覆盖角度 |

## 围栏

基本信息：名称、类型、范围、激活时间、状态、创建时间、应用对象

## 账户管理

管理员/普通账户：用户名、姓名、电话、邮箱、工号、部门、账户权限、修改密码（仅管理员）

新账户默认密码：123456

## 数据分析

| 分类 | 功能 |
|------|------|
| 围栏 | 电子围栏出发统计（时间、围栏-单选） |
| 叉车 | 行驶里程、工作时间、运行轨迹图、运行热力图、动态回放、碰撞事故曲线图、叉车使用率曲线图 |
| 驾驶员 | 行驶里程、工作时间、运行轨迹图、运行热力图、动态回放 |
| 告警 | 告警数量统计（时间、类型-可多选） |
| 高级 | 驾驶员评分排名（时间：1-7天） |


# 页面功能清单

- 列表通用：增删改查、排序、筛选
1. 大屏
2. 实时地图：搜索（POI）、路径规划与导航、定位对象、标签、基站、告警列表；热力图（选择时间/对象/权重）；轨迹图（选择时间/对象/权重）
3. 叉车管理：列表、详情、buffer 设置
4. 驾驶员管理：列表、详情、buffer 设置
5. 设备管理：标签管理（定位工牌、叉车告警灯）列表详情；基站管理列表详情
6. 电子围栏管理：静态电子围栏（列表、详情、设置编辑、删除、图形绘制）；动态电子围栏 buffer（见叉车/驾驶员管理）
7. 告警管理：历史告警（时间规则、时间选择、类型多选、主动/自动告警）；当前告警（开始时间查询、类型多选）
8. 数据分析
9. 系统管理：服务器、账户管理（增删改查）、个人账户（注销、改）；账户类型权限：admin（高于管理员不可删改）、管理员（可增删改）、普通（只能看自己）

# 告警文案定义

- 低电量告警：`19# 叉车的标签XXXXX 在 2020-01-09 16:00 电量低于20%`
- 围栏告警：`王建国 在 2020-01-10 15:00 违规进入 围栏4 区域围栏` / `违规超出`
- Buffer 告警：`19# 叉车 进入动态围栏`

## 室内分楼层显示

对比 wrld3d 和蜂鸟方案：
- 室外→室内切换：蜂鸟点击「进入楼宇」进入室内
- 室内→室外切换：按钮切换
- 楼层切换：蜂鸟单击切换（网页端更适合），wrld3d 滑动按钮（适合移动端）
- 建议网页版参照蜂鸟方案，楼层切换参考高德/百度/蜂鸟

# 移动端功能

## 地图基本功能

- 地图显示：室外（卫星图/2D/3D）、室内（2D/3D）、室内分楼层
- 覆盖物：图标、围栏、信息框
- 控制：指北针、缩放、室内外切换、楼层切换、地图模式切换、路线/导航按钮
- 搜索：基本信息查询、分类查询、公共设施搜索、输入提示、搜索结果列表（按距离排序）、历史记录
- 路径规划：起点终点规划路线、总路程和总时间显示、进入导航按钮
- 导航：导航路径、路径偏移重新规划、语音播报、导航视角、路线文字描述

## 移动端导航讨论

1. 是否需要工作单列表、账户登录
2. 推车上有多个需要入库的对象时，入库路线如何安排
3. 是否需要小铃铛信息提醒
4. 工作单完成后是否需要提交「已完成」反馈
5. 是否要独立搜索功能
