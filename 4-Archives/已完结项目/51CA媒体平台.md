# 项目概述

加拿大最大华人媒体平台之一，负责前后端开发和移动端开发。

# 技术栈

- 后端：PHP / Laravel / Lumen
- 前端：React / jQuery
- 移动端：Android（原生）
- 数据库：MySQL
- 部署：AWS

# 项目模块

- 主站 51.ca
- 问吧 wenba.ca（社区问答）
- Touch版（移动端网页）
- 新闻App（Android/iOS）
- 团购 tuan.51.ca
- 房产 house.51.ca / 51home.ca
- 微信公众号/小程序

# React组件开发流程

使用 rollup 监听 `src/` 组件变化，自动编译到 `dist/`。
example 目录下通过 `npm link` 关联组件进行本地调试。
