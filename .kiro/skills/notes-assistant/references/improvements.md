# User Preferences & Improvement Records

This file tracks user corrections and preferences to personalize the assistant's behavior over time. Read this file at the start of each conversation. Update it when the user provides feedback.

## How to Use This File

When the user says things like:
- "以后不要问我这个" → Record as a sync/workflow preference
- "记住我的偏好" → Record the specific preference
- "这样做不对" → Record the correction with correct behavior
- "查看改进记录" → Show the contents of this file
- "修改/删除第N条" → Edit this file accordingly

## Record Format

Each record should follow this structure:

```markdown
### YYYY-MM-DD: Brief description

**问题**: What went wrong or what the user wants changed
**正确做法**: The correct behavior going forward
**适用场景**: When this rule applies
**示例**: Concrete example
```
