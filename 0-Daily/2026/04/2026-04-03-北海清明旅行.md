# 🏖️ 北海清明旅行（4.3 - 4.6）

> 4月3日（周五）下午出发 → 4月6日（周一）下午返回，共4天3晚

## 📅 行程总览

| 日期 | 主题 | 核心安排 |
|------|------|----------|
| 4/3 周五 | 出发 + 北海初体验 | 下午出发 → 抵达北海 → 北海老街夜游 |
| 4/4 周六 | 涠洲岛（上岛） | 早班船上岛 → 鳄鱼山火山公园 → 石螺口海滩日落 |
| 4/5 周日 | 涠洲岛 + 回北海 | 五彩滩日出 → 天主教堂 → 彩色公路 → 17:00船回北海 |
| 4/6 周一 | 北海市区 + 返程 | 银滩 → 冠头岭/红树林 → 侨港风情街午餐 → 下午高铁回广州 |

---

## 🗓️ 详细行程

### Day 1（4/3 周五）— 出发 + 北海老街

**下午：高铁出发前往北海**
- 🚄 D3738 广州南 14:45 → 北海 19:53（约5小时）
- 大人和小朋友均需请周五下午半天假
- 北海站打车到市区/老街约15-20分钟

**晚上：北海老街夜游**
- 🏛️ 北海老街（珠海路）— 百年骑楼建筑，适合拍照散步
- 🍜 晚餐推荐：老街附近吃虾饼、猪脚粉、糖水
- ⚠️ 老街珍珠和海鲜干货价格虚高，不建议购买

**住宿：** 北海市区（银滩附近或老街附近）

---

### Day 2（4/4 周六）— 涠洲岛上岛

**早上：坐船上涠洲岛**
- 🚢 北海国际客运港 → 涠洲岛（约70分钟）
- ⚠️ 船票需提前在"来游吧"官方平台预订，清明假期至少提前3天抢票
- 💡 选"北游25/26"大船，B区座位较平稳，减少晕船
- 🎫 上岛门票：98元/人（含鳄鱼山景区），1.2m以下儿童免票

**上午：入住 + 休整**
- 入住岛上民宿/酒店（南湾附近方便吃海鲜）

**下午：鳄鱼山火山地质公园**
- 🌋 涠洲岛核心景点，火山岩地貌壮观
- 适合带小朋友了解地质知识
- 游览约2-3小时

**傍晚：石螺口海滩看日落**
- 🌅 涠洲岛最佳日落观赏点
- 沙滩细软，适合小朋友玩沙
- 可以体验潜水（自费项目）

**晚上：南湾海鲜市场**
- 🦐 买海鲜找附近餐馆加工，性价比高
- 推荐：濑尿虾、石斑鱼、扇贝

**住宿：** 涠洲岛（南湾或滴水村附近）

---

### Day 3（4/5 周日）— 涠洲岛 + 回北海

**早上：五彩滩看日出（可选）**
- 🌄 日出时间约6:00，需早起
- 退潮时海蚀平台露出，色彩斑斓
- 小朋友可以在潮间带捡贝壳、看小螃蟹

**上午：涠洲盛塘天主教堂**
- ⛪ 清同治年间法国传教士建造，哥特式建筑
- 免费参观，拍照很出片
- 游览约30-40分钟

**中午前：彩色公路骑行**
- 🚲 租电动车或自行车，沿海岸线骑行
- 沿途风景优美，适合一家人慢慢逛
- 途经贝壳沙滩、圣母堂等小景点

**午餐：南湾或岛上餐馆**
- 最后一顿岛上海鲜，吃完收拾退房

**下午：滴水丹屏 → 坐船回北海**
- 💧 滴水丹屏逛一下，火山岩壁上水滴落的独特景观
- 🚢 17:00 涠洲岛 → 北海（约70分钟），约18:10到北海

**晚上：北海市区**
- 🍜 晚餐可以去侨港风情街或银滩附近觅食
- 饭后银滩散步，感受夜晚海风

**住宿：** 北海市区（银滩附近）

---

### Day 4（4/6 周一）— 北海市区 + 返程

**上午：北海银滩**
- 🏖️ "天下第一滩"，沙质细白如银
- 免费开放，小朋友可以玩沙踏浪
- 上午人少体验更好，游玩约1.5-2小时

**上午（备选）：冠头岭国家森林公园 或 金海湾红树林**
- 🌿 冠头岭：海边礁石，适合拍照
- 🌳 红树林：赶海体验，小朋友会很喜欢（需看潮汐）
- 二选一，根据兴趣和时间决定

**中午：侨港风情街午餐**
- 🍜 北海美食聚集地
- 推荐：卷粉、糖水、越南春卷、虾饼
- 价格实惠，种类丰富
- ⏰ 12:30前吃完，准备去车站

**下午：高铁返回广州**
- 🚄 D3724 北海 14:45 → 广州南 20:00（约5小时15分钟）
- 需要在14:00前到北海站，侨港风情街距北海站约15分钟车程

## ⚠️ 注意事项

1. **涠洲岛船票** — 清明假期非常紧张，务必提前在"来游吧"官方平台抢票，不要找黄牛
2. **防晒** — 4月北海紫外线已经很强，一定要涂防晒霜
3. **海鲜** — 第一顿别吃太多，让肠胃适应；小朋友注意过敏
4. **交通** — 北海市区打正规网约车或出租车，不坐非正规车辆
5. **涠洲岛交通** — 岛上租电动车最方便，约60-80元/天
6. **天气** — 关注天气预报，如遇大风浪船可能停航，需有备选方案
7. **潮汐** — 五彩滩需退潮时去才好看，提前查潮汐表
