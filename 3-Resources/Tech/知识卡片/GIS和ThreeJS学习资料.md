# GIS 和 ThreeJS 技术栈

## 基础知识
- 瓦片/矢量瓦片、GeoJson

## WEB 前端地图框架
- OpenLayers
- mapbox-gl-js

## Geo 可视化
- turf.js、supercluster.js

## 3D + GIS
- webgl-globe（纹理）
- CesiumJS（tile 偏可视化）
- ArcGIS JS（tile 偏 GIS）
- ol-cesium（tile 还不成熟）

## GL 大数据
- deck.gl（Uber）
- mapv（百度）

## 底图
- 高德、Bing、Mapbox、OSM

## 数据库
- PostgreSQL + PostGIS

## 服务端渲染
- mapnik、mapbox-gl-native、tilestrata、tilelive、tippecanoe

## 招聘参考（ThreeJS 岗位）

职责：
1. 主导 3D 高精地图产品开发，基于 ThreeJS 实现 3D 模型网页端展示
2. 基于 WebGL 的 3D 建筑模型生成、展示、全景及交互操作
3. 模型渲染效果优化，提升加载效率和显示性能
4. 优化移动端显示性能

要求：
1. 2 年以上 WebGL 开发经验
2. Three.js 开发经验，计算机图形学基础
3. 熟悉 React/Vue、ES6/TypeScript
4. GIS 领域相关专业知识
