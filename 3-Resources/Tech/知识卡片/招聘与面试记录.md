# 招聘 JD

## 后端工程师

1. 计算机或相关专业本科及以上学历，2 年以上相关行业经验
2. Java 基础扎实，理解 JVM 原理并有一定调优经验，有多线程、高并发系统开发经验优先
3. 熟悉主流开发框架：Spring、SpringBoot、MyBatis、Redis、SpringCloud、ZooKeeper、Netty 等
4. 熟悉分布式、缓存、异步等机制，有分布式系统、集群架构设计和使用经验优先
5. 熟悉 MySQL、MongoDB、Postgres 等数据库，具备一定数据库调优经验
6. 熟悉消息队列 RabbitMQ 应用技术
7. 良好的沟通能力，较强的逻辑思维能力
8. 具有室内定位/煤矿/隧道/地铁/电站/智能制造等相关行业开发经验优先

## 测试工程师

职责：
1. 负责产品全流程测试（需求分析、测试策略、测试计划、测试用例）
2. 负责 IoT 软件/硬件测试（功能、接口、性能、兼容性）
3. 执行测试用例，提交 BUG，跟踪回归测试
4. 分析并推动解决测试问题
5. 使用/开发自动化测试工具

要求：
1. 大专及以上学历，计算机/电子/自动控制相关专业
2. 掌握基本测试理论和流程，熟悉 IPD 或敏捷开发流程优先
3. 有物联网软件/硬件测试经验优先
4. 熟悉 JMeter 等性能测试工具

### 2014 微软室内定位比赛

研究生原来的研究方向是加密算法，但在室友邀请下加入了室内定位方向的研究。2014 年跟室友（现在的 CEO）一起参加在美国西雅图举行的微软室内定位比赛。因预算超标系主任不支持，最终自己掏钱成行。取得了不错的名次，看到了行业领先技术，积累了人脉。这些人脉在后续创业中陆续浮现。
